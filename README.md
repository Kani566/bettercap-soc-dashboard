# 🛡️ Bettercap SOC Dashboard

A network monitoring and security operations setup built on Kali Linux using Bettercap.

## 📌 Project Overview
This project demonstrates a SOC (Security Operations Center) dashboard for real-time network monitoring, device discovery, and ARP-based traffic analysis in a controlled lab environment.

## 🛠️ Tools & Technologies
- Kali Linux
- Bettercap v2.41.5
- VirtualBox
- Firefox

## ⚙️ Features
- Real-time LAN device discovery
- Live network traffic sniffing
- ARP spoofing simulation
- DNS request monitoring
- Web-based SOC dashboard

## 🚀 Setup Instructions
1. Install Bettercap
2. Install bettercap-caplets
3. Download UI files
4. Launch dashboard

## ⚠️ Disclaimer
This project is for educational purposes only. Only use on networks you own or have permission to test.
